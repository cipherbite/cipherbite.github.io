DNS Exfiltration Detector

Real-time Detection of Data Theft via DNS

Overview

Python-based tool that detects data exfiltration attempts through DNS queries. Analyzes DNS traffic for suspicious patterns including high entropy domains, Base64 encoded data, and abnormal query frequencies.

Python Scapy Pandas Wireshark

How It Works

The detector monitors network traffic and analyzes DNS queries in real-time.

[Screenshot: Terminal showing dns_detector.py running with colored output]
$ sudo python3 dns_detector.py -i eth0
[+] DNS Exfiltration Detector started
[*] Monitoring interface: eth0
[*] Entropy threshold: 4.0

Detection Methods

Entropy Analysis

Calculates Shannon entropy to identify random/encoded domain names

Pattern Matching

Detects Base64, Hex encoding, and known exfiltration patterns

Frequency Analysis

Monitors query rates to identify DNS tunneling behavior

Core Detection Logic

Python implementation using Scapy for packet capture and analysis.

[Screenshot: VS Code with dns_detector.py open showing entropy calculation function]

The entropy calculation identifies domains with high randomness - a key indicator of encoded data.
