Core Detection Algorithm

# DNS Entropy Analysis for Exfiltration Detection
import math
from collections import Counter

def calculate_entropy(domain):
    """Calculate Shannon entropy of a domain name"""
    # Remove TLD for analysis
    subdomain = domain.split('.')[0]
    
    # Calculate character frequency
    freq = Counter(subdomain)
    length = len(subdomain)
    
    # Shannon entropy calculation
    entropy = -sum((count/length) * math.log2(count/length) 
                   for count in freq.values())
    
    return entropy

def detect_suspicious_dns(query):
    """Detect potential DNS exfiltration"""
    entropy = calculate_entropy(query.domain)
    
    # High entropy indicates possible encoded data
    if entropy > 3.5 or len(query.domain) > 50:
        return {
            'threat': True,
            'entropy': entropy,
            'action': 'BLOCK'
        }
    
    return {'threat': False}

How it works:

The algorithm calculates the Shannon entropy of DNS queries. Normal domain names have low entropy (around 2.5), while base64 or hex-encoded data has much higher entropy (>3.5). This helps identify DNS tunneling attempts where attackers encode stolen data in subdomain names.

Machine Learning Model

# LSTM Neural Network for DNS Pattern Recognition
from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import LSTM, Dense, Dropout

def build_detection_model():
    """Build LSTM model for DNS sequence analysis"""
    model = Sequential([
        LSTM(128, return_sequences=True, input_shape=(100, 38)),
        Dropout(0.2),
        LSTM(64),
        Dropout(0.2),
        Dense(32, activation='relu'),
        Dense(1, activation='sigmoid')
    ])
    
    model.compile(
        optimizer='adam',
        loss='binary_crossentropy',
        metrics=['accuracy']
    )
    
    return model

Machine Learning Approach:

The LSTM (Long Short-Term Memory) neural network analyzes sequences of DNS queries to detect patterns indicative of data exfiltration. It's trained on millions of both legitimate and malicious DNS queries, learning to distinguish between normal traffic and covert data channels.
